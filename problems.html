JavaScript Problem Set

You can use these problems as part of your project or as standalone coding tasks. Try solving them in the Online Code Playground.

Problem 1: Unique Values Counter

Given an array of numbers, return the number of unique values.

Example:

// Input:
[1, 2, 3, 2, 4, 3, 5]

// Output:
5

Hint: You can use a Set in JavaScript.

Problem 2: Student Topper

You are given an array of student objects, each having name and marks.
Return the name of the student who has the highest marks. 

// Input:
[
  { name: "Arjun", marks: 78 },
  { name: "Bhavya", marks: 92 },
  { name: "Chitra", marks: 85 }
]

// Output:
"Bhavya"

Problem 3: String Compression

Given a string consisting of lowercase letters, compress it by replacing each group of repeating characters with the character followed by the count of repetitions.

Example:

// Input:
"aaabbc"

// Output:
"a3b2c1"

Problem 4: Async Fetch Wrapper (Advanced)

Write an async function fetchJson(url) that:
• fetches data from the given URL
• returns parsed JSON on success
• throws an error with message "Request failed" if response is not ok 

// Example usage:
fetchJson("https://api.example.com/data")
  .then(data => console.log(data))
  .catch(err => console.error(err.message));
